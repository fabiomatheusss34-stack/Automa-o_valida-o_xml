# Validador XML — Grupo Ecoa x T. Globo

Sistema em Python para validação de XMLs de NF-e, comparação com Purchase Order (PO) e análise de recebimentos.

## Funcionalidades

* Leitura de XML NF-e
* Extração de SKU, EAN, quantidade e valores
* Validação interna do XML
* Comparação XML x PO
* Validação de EAN
* Comparação de preços
* Análise de recebimento
* Geração automática de relatório Excel

## Tecnologias utilizadas

* Python
* Pandas
* OpenPyXL
* Tkinter

## Instalação

Clone o projeto:

```bash
git clone https://github.com/fabiomatheuss34-stack/Automa-o_valida-o_xml.git
```

Instale as dependências:

```bash
pip install -r requirements.txt
```

Execute:

```bash
python validar_xml_globo_v2.4.py
```
